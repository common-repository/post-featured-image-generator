Parse post content and automatically generate post featured image.

== Description ==

<h3>Features:</h3>
<ul>
<li>Min image size filters</li>
<li>Parsing priority</li>
<li>Post types filters (inc. custom post types)</li>
<li>Old post processing via bulk regeneration (with options)</li>
</ul>

<h3>Supports:</h3>
<ul>
<li>Direct links to images</li>
<li>Youtube</li>
</ul>

== Installation ==

1. Upload archive content to the `/wp-content/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
